Proyek Saya

Proyek CRUD Pendataan Barang dengan Laravel

Sebuah aplikasi web fungsional untuk mengelola data inventaris barang. Proyek ini menerapkan operasi CRUD (Create, Read, Update, Delete) penuh, yang dibangun menggunakan framework Laravel untuk memproses form, validasi data, dan interaksi dengan database MySQL.

Aplikasi Pemesanan Pizza Berbasis Konsol

Proyek awal yang dirancang untuk mengimplementasikan logika pemrograman dasar menggunakan Python. Aplikasi berbasis konsol ini berfungsi sebagai simulasi sistem kasir, memandu pengguna dari pemilihan menu (topping, crust, size) hingga rekapitulasi pesanan akhir. Proyek ini menonjolkan kemampuan dalam mengelola alur program, validasi input pengguna, dan melakukan kalkulasi berdasarkan pilihan yang kompleks.

Program Pencari Rute Terpendek Dengan Mengimplementasikan Algoritma Dijkstra & TSP

Proyek dari mata kuliah Struktur Data yang mengaplikasikan teori Graf untuk pemecahan masalah pencarian rute. Aplikasi konsol ini memiliki dua fitur utama: (1) Menemukan jalur terpendek antara dua kota menggunakan Algoritma Dijkstra, dan (2) Mencari rute tur paling efisien yang mengunjungi setiap kota tepat satu kali, sebuah implementasi dari Travelling Salesman Problem (TSP). Program ini menunjukkan kemampuan dalam menerapkan algoritma kompleks dan struktur data dalam Python.
